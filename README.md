# уровень 17

мобильная party-игра для 17-летия сони. открывается по ссылке или qr-коду, без установки, аккаунтов и сервера.

## запуск локально

откройте `index.html` в браузере или запустите любой статический сервер в этой папке.

## публикация

подойдёт любой статический хостинг:

- github pages
- netlify
- vercel
- cloudflare pages

после публикации сделайте qr-код на публичную ссылку и отправьте его гостям.

## формат партии

- 9-14 человек
- 4 команды: ловцы покемонов, вайнеры последнего дня, орден musical.ly, секта дэба
- 5 раундов по умолчанию
- 15-20 минут
- каждый телефон хранит только свой локальный счёт
